    🪪 Gerador de Autorização de Crachá SAH
Um sistema web simples e prático desenvolvido para automatizar a geração e impressão de documentos de autorização para a confecção de crachás de funcionários. O projeto é voltado para uso administrativo, facilitando o preenchimento dos dados e a emissão do formulário padrão.

🌍 Acesse a aplicação aqui: [Gerador de Autorização de Crachá](https://zinnxs.github.io/Cracha/)

    📋 Sobre o Projeto
O "Gerador de Autorização" foi criado para otimizar o processo de solicitação de crachás (1ª e 2ª via). Através de uma interface limpa, o usuário insere os dados do colaborador e o sistema gera automaticamente um documento formatado (com cabeçalho institucional, data atual e tabela de assinaturas) pronto para ser impresso.

O cabeçalho gerado atende ao formato da Prefeitura de Volta Redonda e do respectivo Hospital/Instituição (SAH).

    ✨ Funcionalidades
Formulário Dinâmico: Campos para preenchimento de Matrícula, Nome do Funcionário e Função.

Seleção de Via: Opção para escolher se a autorização é para a 1ª via ou 2ª via do crachá.

Geração de Documento: 
°Criação instantânea de um layout pronto para impressão contendo:

°Cabeçalho institucional.

°Data formatada automaticamente.

°Tabela organizada com os dados preenchidos e espaço para assinatura.

°Controle de Impressão: Botões dedicados para imprimir o documento gerado (🖨️ Imprimir Agora) ou retornar para corrigir os dados (⬅ Voltar e Editar).

    🛠️ Tecnologias Utilizadas
Este projeto é uma aplicação Front-end estática hospedada no GitHub Pages, utilizando tecnologias web padrão:

HTML5: Estruturação semântica da página e do documento de impressão.

CSS3: Estilização da interface de usuário e formatação específica para a folha de impressão (Print CSS).

JavaScript (Vanilla): Captura dos dados do formulário, manipulação do DOM para exibir o documento e integração com a função de impressão do navegador.

    🚀 Como Usar
Acesse o link da aplicação.

Na tela inicial (Gerador de Autorização), preencha os seguintes dados:

Matrícula do funcionário.

Nome completo do funcionário.

Função (cargo).

Selecione a via do crachá desejada (1ª via ou 2ª via).

Clique no botão "Gerar Documento".

Na tela de visualização do documento:

Verifique se os dados estão corretos.

Para imprimir, clique em "🖨️ Imprimir Agora".

Se precisar corrigir algo, clique em "⬅ Voltar e Editar".
